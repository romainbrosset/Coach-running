# BosVUT-7Q91

PWA statique privée par obscurité pour le suivi running, trail, feedback séance et calcul nutrition Näak.

## Utilisation

Ouvre `index.html` localement ou publie le dossier sur GitHub Pages. L’app démarre avec l’historique Garmin préchargé depuis `data/seed-data.js`, stocke les données dans le navigateur et propose un export JSON depuis l’onglet Données.

## Confidentialité

GitHub Pages ne protège pas un site public avec une authentification native. Le nom discret, `robots.txt` et `noindex` réduisent la découverte, mais ne remplacent pas un vrai contrôle d’accès. Évite de publier les fichiers Garmin bruts dans un dépôt public si tu veux garder ces données privées.

## Déploiement GitHub Pages

Un nom de dépôt discret cohérent avec l’app serait `bosvut-7q91`. Publie la branche contenant ces fichiers depuis GitHub Pages, puis importe les fichiers Garmin depuis ton navigateur.
